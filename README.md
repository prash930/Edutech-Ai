# 🎓 EduPath AI — Study Abroad Platform

AI-powered student engagement platform built for the Poonawalla Fincorp Hackathon.

## Features
- 🤖 AI Mentor Chatbot (Groq + LLaMA 3.3 70B)
- 📊 ROI Calculator with charts
- 🎯 AI Admission Predictor
- 💰 Education Loan Funnel (3-step)

## Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the app
```bash
streamlit run app.py
```

### 3. Open in browser
```
http://localhost:8501
```

### 4. Enter your Groq API key
- Get free key at: https://console.groq.com
- Enter it in the sidebar of the app

## Tech Stack
- **Frontend**: Streamlit
- **AI**: Groq API (LLaMA 3.3 70B)
- **Charts**: Plotly
- **Data**: Pandas
